== Description ==

Adds a sidebar widget and display OPML, ATOM, RSS feed link for your WordPress blog.
== Installation ==

1. Upload `syndication-widget` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Syndication Widget to your sidebar

== Frequently Asked Questions ==
= What does this plugin do, exactly? =
It provides feed link of OPML, ATOM, RSS
